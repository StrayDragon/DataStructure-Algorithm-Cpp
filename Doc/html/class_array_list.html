template<typename E >
bool ArrayList< E >::insert ( int  newPosition,
const E &  element 
)
overridevirtual

向指定位置插入元素 指定位置的最大有效值定义为:元素数量+1
若指定位置有效
若此位置不存在元素,将会在此位置插入元素
若此位置存在元素,将会在此位置之前插入元素,维持逻辑位置与指定位置相同
成功插入后,列表中的内容将重新排号.


例:

//... Initialize aList
//aList.insert(0,""); // Violated by post condition
aList.insert(1,"Cpp"); // ["Cpp"]
aList.insert(1,"Modern") // ["Modern","Cpp"]
aList.insert(3,"!") // ["Modern","Cpp","!"]
后置条件
1 <= newPosition <= getLength() + 1
参数
newPosition插入的位置,这里从列表的第一项开始计数
element待插入元素
返回
true 插入成功
false 插入失败
